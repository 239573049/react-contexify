Features​

  • Easy to set up for real, you can make it work in less than 10sec!
  • Super easy to customize thanks to css variables 💅
  • Custom position
  • Sub menu support
  • Does not go offscreen
  • Dark mode 🌒
  • Keyboard navigation + keyboard shortcut!
  • Built-in animations
  • Easy to test!
  • Written in Typescript 💪
  • Tiny! (3k gzipped)
